BẢNG ÁNH XẠ URL - WEBSITE SACHQUOCGIA.VN

Ngày thu thập: 26/02/2026 | Phương pháp: HTTP Request + Sitemap Crawling



Ghi chú kỹ thuật

Website sachquocgia.vn là Single Page Application (SPA). Các trang tài khoản/đăng nhập/tìm kiếm sử dụng JavaScript client-side routing, do đó không trả về nội dung khi fetch trực tiếp qua HTTP. URLs được thu thập qua: (1) Sitemap XML, (2) Phân tích HTML từ HTTP requests, (3) Phân tích cấu trúc URL patterns.

Chú thích

[SPA] = Single Page Application routing - URL chỉ hoạt động khi mở trên trình duyệt có JavaScript. Không thể fetch trực tiếp bằng HTTP.

[Xác nhận] = URL trả về HTTP 200 OK khi fetch trực tiếp.

Phương pháp thu thập: (1) Đọc Sitemap XML tại /sitemap.xml, (2) Phân tích HTML từ HTTP requests, (3) Phân tích URL patterns từ navigation/footer/links.
